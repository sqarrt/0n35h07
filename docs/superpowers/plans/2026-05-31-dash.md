# Dash Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Добавить рывок (Shift при зажатом WASD) — короткий бросок ~0.15с в направлении движения, с кулдауном, реакцией FOV и отменой заряженного выстрела.

**Architecture:** Intent-метод `Player.dash(dir)` (вшитый кулдаун) делегирует в `Body` (состояние рывка → копит в `desired`, который применяет KCC в `Match.applyPhysics`). `HumanController.onDash` считает WASD-направление; `Game.tsx` вешает Shift. Успешный рывок зовёт `IWeapon.interrupt()` — отменяет windup в cooldown.

**Tech Stack:** TypeScript, Three.js, @react-three/rapier (KCC), vitest (юнит), Playwright (e2e).

(3c) Методы — рядом с `stepVertical`:
```ts
  /** Старт рывка: true если кулдаун готов и направление ненулевое. */
  dash(dir: THREE.Vector3): boolean {
    if (this.dashCooldown > 0) return false
    this.dashDir.set(dir.x, 0, dir.z)
    if (this.dashDir.lengthSq() === 0) return false
    this.dashDir.normalize()
    this.dashTimer = DASH_DURATION
    this.dashCooldown = DASH_COOLDOWN
    return true
  }

  /** Копит рывок в desired и тикает таймеры (зовётся из Match.applyPhysics). */
  stepDash(dt: number) {
    if (this.dashCooldown > 0) this.dashCooldown -= dt * 1000
    if (this.dashTimer > 0) {
      this.desired.addScaledVector(this.dashDir, DASH_SPEED * dt)
      this.dashTimer -= dt * 1000
    }
  }

  get dashing() { return this.dashTimer > 0 }
```

import { unlockPointer, holdKey, getCameraPos } from './helpers'

async function release(page: any, code: string) {

  const n0 = await getCameraPos(page)
  await holdKey(page, 'KeyW', 400)

  const d0 = await getCameraPos(page)
  await press(page, 'KeyW')
