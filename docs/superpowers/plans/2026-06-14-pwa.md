# PWA Install Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Добавить PWA-установку игры через браузер — кнопка в адресной строке Chrome/Edge, окно без chrome.

**Architecture:** `vite-plugin-pwa` генерирует манифест и пустой SW (нет кэша — только регистрация). Иконки нарезаются из `build/icon.png` через `@vite-pwa/assets-generator`. SW не влияет на геймплей.

**Tech Stack:** `vite-plugin-pwa`, `@vite-pwa/assets-generator` (one-shot CLI)

---

### Task 1: Сгенерировать PWA-иконки

**Files:**
- Create: `public/pwa-192x192.png`
- Create: `public/pwa-512x512.png`

- [ ] **Шаг 1: Запустить генератор иконок**

```bash
npx @vite-pwa/assets-generator@latest --preset minimal --source build/icon.png
```

Ожидаемый вывод: несколько строк вида `✔ Generated public/pwa-192x192.png` и т.д.

- [ ] **Шаг 2: Проверить что файлы созданы**

```bash
ls public/pwa-*.png
```

Ожидаемый результат — минимум два файла: `pwa-192x192.png` и `pwa-512x512.png`.

- [ ] **Шаг 3: Зафиксировать иконки**

```bash
git add public/pwa-192x192.png public/pwa-512x512.png
git commit -m "feat(pwa): иконки 192 и 512px"
```

### Task 2: Установить и сконфигурировать vite-plugin-pwa

**Files:**
- Modify: `vite.config.ts`

- [ ] **Шаг 1: Установить пакет**

```bash
npm install -D vite-plugin-pwa
```

- [ ] **Шаг 2: Обновить vite.config.ts**

Заменить содержимое файла на:

```ts
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import { VitePWA } from 'vite-plugin-pwa'
import { editorMaps } from './vite-plugin-editor-maps'
import { cameraPoses } from './vite-plugin-camera-poses'
import pkg from './package.json'

export default defineConfig({
  plugins: [
    react(),
    editorMaps(),
    cameraPoses(),
    VitePWA({
      registerType: 'autoUpdate',
      manifest: {
        name: '0N35H07',
        short_name: '0N35H07',
        description: 'Аркадный шутер от первого лица, строго 1v1 (p2p)',
        theme_color: '#000000',
        background_color: '#000000',
        display: 'standalone',
        start_url: '.',
        icons: [
          { src: 'pwa-192x192.png', sizes: '192x192', type: 'image/png' },
          { src: 'pwa-512x512.png', sizes: '512x512', type: 'image/png' },
        ],
      },
      workbox: {
        globPatterns: [],
      },
    }),
  ],
  base: './',
  define: {
    __APP_VERSION__: JSON.stringify(pkg.version),
  },
  server: {
    port: 5173,
    strictPort: true,
    watch: {
      ignored: ['**/menuCameraPoses.json'],
    },
  },
})
```
