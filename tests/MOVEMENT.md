# MOVEMENT — Сценарий тестирования передвижения

## Цель
Убедиться что персонаж движется по арене, прыгает и не проваливается сквозь пол.

## Шаги

### 1. Начальная позиция
- Открыть `http://localhost:5173`
- Скриншот: персонаж стоит перед мишенью, пол и стены видны

### 2. Движение вперёд (W)
- Через `browser_evaluate`: прочитать `camera.position` до и после нажатия W
- Ожидается: Z-координата уменьшилась (движение к мишени)

### 3. Движение назад (S)
- Ожидается: Z-координата увеличилась

### 4. Стрейф влево/вправо (A/D)
- Ожидается: X-координата изменилась

### 5. Прыжок (Space)
- Нажать Space, сделать скриншот в момент прыжка
- Ожидается: камера поднялась выше EYE_HEIGHT (1.7)
- После приземления: Y возвращается к 1.7

### 6. Не проваливается сквозь пол
- Ожидается: Y никогда не опускается ниже 1.7

## Проверка через playwright-mcp
```js
// Прочитать позицию камеры из сцены R3F
const pos = await page.evaluate(() => {
  // R3F хранит renderer в __r3f на canvas
  const canvas = document.querySelector('canvas')
  return canvas?.__r3f?.fiber?.getState?.()?.camera?.position
})
```
