# SHIELD — Сценарий тестирования щита

## Цель
Убедиться что щит активируется по ПКМ, визуально отображается, имеет кулдаун.

## Шаги

### 1. Щит активируется по ПКМ
- ПКМ → скриншот сразу после нажатия
- Ожидается: синее кольцо в HUD (вокруг прицела), синяя сфера в 3D-сцене

### 2. Щит исчезает через 500ms
- Скриншот через 600ms после ПКМ
- Ожидается: кольцо и сфера исчезли

### 3. Кулдаун после щита
- После деактивации: бар `SHIELD [ПКМ]` опустел и заполняется
- Пока кулдаун (~2000ms) — повторный ПКМ не активирует щит
- Ожидается: через ~2500ms от первого нажатия бар снова полный

### 4. Контекстное меню заблокировано
- ПКМ не должен открывать контекстное меню браузера
- Ожидается: никакого popup

### 5. Щит не мешает стрельбе
- Активировать щит и сразу нажать ЛКМ
- Ожидается: луч вылетает (механики независимы)

## Проверка через playwright-mcp
- Скриншот сразу после ПКМ: ищем синее кольцо в HUD
- Скриншот через 700ms: кольцо исчезло
- Консоль: 0 errors, нет ошибок pointer lock
